Les liens hypertextes

L'anatomie d'un lien

On crée un lien en utilisant la balise a (anchor => ancre). Cette balise a besoin d'attributs pour fonctionner correctement. Ces attributs définissent :

  • Le document lié (page html, fichier, email, un numéro de téléphone, etc.
  • La façon dont on ouvre le document lié (elle-même, un autre onglet, etc.)
  • Un titre de lien, qui "explique" où mène le lien

Les attributs en détail

Le chemin vers le document ciblé : href

C'est le chemin vers le document à partir de la page sur laquelle se trouve le lien.

le dossier "frame"

href peut aussi contenir l'identifiant d'un élément de la même page, en cliquant sur le lien on ammène en haut de l'affichage l'élément identifié, cela est souvent utilisé pour créer une table des matières en début d'un long document.

Remonter au début de la partie "Attributs"

Cible où sera ouverte le document ciblé : target

target défini la cible d'ouverture du document ciblé. la coble est un cadre (frame)

par exemple, _self (configuration par defaut), fait ouvrir dans le cadre actuel, _blank ouvre la page dans un nouvel onglet, _parent : ouvre le lien dans un cadre parent.
